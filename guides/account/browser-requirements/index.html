Browser Requirements

Basecamp is built on the nifty features found in modern browsers. You'll need to use one of the browsers below to use Basecamp:

Internet Explorer 9+

Google Chrome - latest version

Mozilla Firefox - latest version

Apple Safari 5+

By the way: Basecamp may not work properly with beta or pre-release versions of these browsers. Download the latest commercial version above for the best experience.
