Archiving, Deleting and Renaming projects

Your Project Settings page lets you archive, unarchive, or delete a project.

  1. Archive or delete a project: When a project is over, you can archive it for safe keeping. Archived projects can’t be modified and they don’t count against your project number limit (although files in them do count toward your file storage limit). Everything stays just as you left it, and you can unarchive a project at any time. If you want a project removed entirely, you can delete it. Scroll to the bottom of your project and click "Project settings."

    Select the "Archived" option then save changes to make it official. Or click the delete link to remove it. Don't worry; you can restore a deleted project for 30 days!

    To archive or delete a project, you'll need to either be an admin or the person who created the project.

  2. Looking for your archived projects? Look for the link on your Projects page.

    Email reminders about future events & tasks will not be sent from archived projects. Basecamp will still add any items sent via email to your archived project.

  3. Renaming a project: Click into the project then on the name of the project in the upper left corner. It’ll turn into an editable text field. Now you can change the project’s name! Make any changes you’d like to the name or description, then click the Save changes button.
