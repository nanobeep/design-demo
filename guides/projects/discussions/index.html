Discussions

Any time a comment is added to a message, to-do, file, or event, a discussion thread will be created. The most recently commented-on threads will display at the top of a project's main page.

Create a stand-alone discussion

  1. Go to a project and click to post a new message.
  2. Type your message, select the notifications, and post it!

Add a link to any discussion by copying and pasting the URL. It becomes clickable when you publish the post. If it's a link to an image or web video, the image or video will automatically be embedded into your discussion. Boom!

Create a discussion about a To-do item

  1. Click on a to-do item within a list.
  2. Click the text box to start your comment and post as usual!

Create a discussion about a file

  1. Go to a specific file and click to discuss it.

  2. Click the text box and say something about the file.

Create a discussion about an event

  1. Go to the Calendar page and click on a specific event, then click the "Discuss this event ... " link.

  2. Click the text box to add a comment on that event.
